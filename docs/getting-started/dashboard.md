# Dashboard Overview

The dashboard is your command center for managing all pricing and discount features.

## Dashboard Layout

When you open the app, you'll see the main dashboard with several key areas:

### Header Section

- **Store Name**: Your connected Shopify store
- **Current Plan**: FREE, GROWTH, or PROFESSIONAL badge
- **Quick Actions**: Shortcuts to common tasks

### Statistics Cards

Four main metrics displayed at the top:

| Metric | Description |
|--------|-------------|
| **Active Rules** | Number of currently active pricing rules |
| **Total Discounts** | Total discount amount given to customers |
| **Orders Affected** | Number of orders that used a discount |
| **Revenue Impact** | Estimated revenue change from discounts |

### Quick Actions

Buttons for common tasks:

- **Create Rule** - Start a new pricing rule
- **View Analytics** - Open detailed analytics
- **Settings** - App configuration

### Recent Activity

Shows the latest activity in your account:

- New rules created
- Rules activated/paused
- Settings changes
- Customer discount usage

## Dashboard Widgets

### Active Rules Summary

Shows your top-performing rules:

- Rule name
- Status (Active/Paused)
- Usage count
- Revenue generated

### Performance Graph

Visual representation of:

- Discount usage over time
- Revenue trend
- Order volume

### Quick Tips

Contextual tips based on your usage:

- Suggestions for optimization
- Feature recommendations
- Best practices

## Refreshing Data

Dashboard data updates automatically, but you can:

1. Click the **Refresh** icon in any widget
2. Use browser refresh (F5)
3. Navigate away and back

{% hint style="warning" %}
Analytics data may have a 5-15 minute delay for accuracy.
{% endhint %}

## Customizing the Dashboard

While the dashboard layout is fixed, you can:

1. **Collapse sidebar**: Click the hamburger menu for more space
2. **Change language**: Settings > Language
3. **Dark/Light mode**: Follows your Shopify admin preference

## Mobile Access

The dashboard is fully responsive:

- Works on tablets and phones
- Same functionality as desktop
- Touch-friendly navigation
