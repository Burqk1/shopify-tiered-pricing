# Installation

This guide walks you through installing Tiered Pricing Pro on your Shopify store.

## Requirements

Before installing, ensure you have:

- An active Shopify store (Basic plan or higher)
- Store owner or staff account with app installation permissions
- A modern web browser (Chrome, Firefox, Safari, or Edge)

## Installation Steps

### Step 1: Find the App

1. Go to the [Shopify App Store](https://apps.shopify.com)
2. Search for "Tiered Pricing Pro"
3. Click on the app listing

### Step 2: Install the App

1. Click the **Add app** button
2. Review the permissions requested:
   - `read_products` - To display your products
   - `write_products` - To update prices (for AI pricing)
   - `read_orders` - For analytics and tracking
   - `read_customers` - For B2B customer groups
3. Click **Install app**

### Step 3: Authorize Access

You'll be redirected to your Shopify admin to authorize the app:

1. Review the permissions one more time
2. Click **Install** to confirm
3. You'll be redirected to the app dashboard

## Post-Installation

After installation, you'll see the onboarding wizard:

1. **Welcome Screen** - Introduction to the app
2. **First Rule** - Create your first pricing rule
3. **Theme Integration** - Enable the pricing table display
4. **Done!** - Start using the app

{% hint style="info" %}
The onboarding wizard can be accessed again from Settings > Help > Run Setup Wizard
{% endhint %}

## Theme Integration

For pricing tables to display on your storefront, the app automatically:

1. Adds a theme app extension
2. Injects the pricing table widget on product pages
3. Handles cart calculations

{% hint style="warning" %}
Some heavily customized themes may require manual adjustment. Contact support if you don't see pricing tables on product pages.
{% endhint %}

## Verifying Installation

To confirm everything is working:

1. Go to **Dashboard** in the app
2. Check that your store name appears in the header
3. Create a test rule and preview it
4. Visit a product page on your storefront

## Uninstalling

If you need to uninstall:

1. Go to Shopify Admin > Settings > Apps
2. Find "Tiered Pricing Pro"
3. Click **Delete**

{% hint style="danger" %}
Uninstalling the app will remove all your pricing rules and settings. This action cannot be undone.
{% endhint %}
