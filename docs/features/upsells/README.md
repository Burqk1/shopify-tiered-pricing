# Upsells & Cross-sells

Increase your average order value by recommending complementary products, upgrades, and add-ons to customers during their shopping journey.

{% hint style="info" %}
**Plan Required:** GROWTH or PROFESSIONAL
{% endhint %}

## Upsell vs Cross-sell

**Upsell**: Encourage buying a better/more expensive version
- "Upgrade to Pro model for just $50 more"
- "Get the larger size for better value"

**Cross-sell**: Suggest complementary products
- "Customers also bought..."
- "Complete the look with..."

## Creating Upsell Offers

### Step 1: Navigate to Upsells

1. Go to **Upsells** in the sidebar
2. Click **Create Upsell**

### Step 2: Select Trigger Product(s)

Which product(s) trigger the upsell:
- Specific product
- Products in collection
- All products
- Products with specific tag

### Step 3: Select Upsell Product(s)

What to recommend:
- Specific product (upgrade)
- Related products (cross-sell)
- AI-recommended (dynamic)

### Step 4: Configure Display

**Display Type:**
- In-page widget
- Popup/modal
- Slide-in drawer
- Inline suggestion

**Trigger Point:**
- Product page view
- Add to cart
- Cart page
- Checkout
- Post-purchase

### Step 5: Set Discount (Optional)

Incentivize the upsell:
- Percentage off upsell item
- Fixed amount off
- Free shipping when added
- Bundle discount

### Step 6: Activate

1. Review settings
2. Click **Create Upsell**
3. Toggle to **Active**

## AI-Powered Recommendations

### How It Works

AI analyzes:
- Purchase history patterns
- Frequently bought together data
- Customer browsing behavior
- Similar customer purchases

### Benefits

- No manual configuration
- Continuously optimizing
- Personalized per customer
- Higher conversion rates

### Enable AI Recommendations

1. Go to Upsells settings
2. Enable "AI Recommendations"
3. Select data sources
4. AI starts learning immediately

## Upsell Strategies

### Price Anchoring

Show expensive option first:
```
Premium: $199 (Most Popular)
Standard: $99
Basic: $49
```

### Decoy Pricing

Make target option look best:
```
Small: $5 (8oz)
Medium: $5.50 (12oz) ← Best value
Large: $8 (16oz)
```

### Limited Time Upgrade

Create urgency:
```
Upgrade to Pro for 50% off
Only available in next 15:00
```

### Free Shipping Threshold

Cross-sell to reach free shipping:
```
You're $15 away from free shipping!
Add: [Product A - $14.99] [Product B - $16.99]
```

## Best Practices

### 1. Relevance is Key

- Recommend truly complementary items
- Match price range to original product
- Consider customer segment

### 2. Limit Choices

- 2-4 recommendations optimal
- Too many = decision paralysis
- Quality over quantity

### 3. Clear Value Proposition

- Explain why they need it
- Show savings clearly
- Highlight benefits

### 4. Timing Matters

| Stage | Best Offer Type |
|-------|-----------------|
| Product page | Upgrades, bundles |
| Add to cart | Accessories, add-ons |
| Cart | Free shipping push |
| Checkout | Warranties, gift wrap |
| Post-purchase | Complementary, replenishment |

### 5. Don't Be Pushy

- One upsell per stage
- Easy to dismiss
- Don't block checkout flow

### 6. Test and Optimize

- A/B test different products
- Try different placements
- Measure conversion impact

## Troubleshooting

### Upsells not showing

1. Verify upsell is "Active"
2. Check trigger conditions
3. Ensure app embed enabled
4. Test in incognito mode

### Wrong products recommended

1. Review trigger settings
2. Check product associations
3. Verify AI training data
4. Update manual selections

### Low conversion rate

1. Test different products
2. Try different placements
3. Add/adjust discount
4. Improve messaging
