## Key Metrics

### Revenue Metrics

| Metric | Description |
|--------|-------------|
| Total Revenue | All revenue from discounted orders |
| Discount Revenue | Revenue with discounts applied |
| Full-Price Revenue | Revenue without discounts |
| Revenue Lift | Additional revenue attributed to pricing |
| Revenue per Visitor | Average revenue per store visitor |

### Order Metrics

| Metric | Description |
|--------|-------------|
| Total Orders | Orders with pricing rules applied |
| Orders with Discount | % of orders using discounts |
| Average Order Value | Mean order total |
| Items per Order | Average items in cart |
| Conversion Rate | Visitors → Purchases |

### Discount Metrics

| Metric | Description |
|--------|-------------|
| Total Savings | Customer savings given |
| Avg Discount % | Average discount percentage |
| Discount Depth | How deep discounts go |
| Stacking Rate | Discounts combined with codes |

## Best Practices

### 1. Check Analytics Regularly

- Daily: Quick health check
- Weekly: Performance review
- Monthly: Deep analysis

### 2. Focus on Revenue Per Visitor

RPV combines conversion rate and AOV - best single metric for pricing effectiveness.

### 3. Compare Time Periods

Always compare to previous periods to understand trends and seasonality.

### 4. Segment Your Data

Different customer groups behave differently - segment to find insights.

### 5. Act on Insights

Data without action is useless:
- Low-performing rules → Optimize or pause
- High-performing rules → Expand or replicate
- Opportunities → Test with A/B

### 6. Document Learnings

Keep a log of:
- What you tested
- What you learned
- What you changed
- Results achieved

## Troubleshooting

### Data not showing

1. Allow 24-48 hours for data processing
2. Verify rules are active
3. Check date range selection
4. Ensure app tracking enabled

### Numbers don't match Shopify

1. Tiered Pricing tracks discounted orders only
2. Time zones may differ
3. Returns/refunds processing delay
4. Check included order types

### Missing revenue attribution

1. Verify order tagging
2. Check discount code tracking
3. Review Shopify Functions logs
4. Contact support if persistent
