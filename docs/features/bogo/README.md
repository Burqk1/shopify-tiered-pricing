# BOGO Offers

**Plan Required:** GROWTH or higher

BOGO (Buy One Get One) offers are powerful promotions that encourage customers to buy more by offering free or discounted items.

## What is BOGO?

BOGO offers come in several flavors:

| Type | Example |
|------|---------|
| Buy X Get Y Free | "Buy 2, Get 1 Free" |
| Buy X Get Y % Off | "Buy 1, Get 2nd 50% Off" |
| Buy X Get Y $ Off | "Buy 3, Get 4th $10 Off" |
| Spend X Get Y | "Spend $100, Get Free Gift" |

## Why Use BOGO?

### Benefits

- **Increases Average Order Value** - Customers buy more to get the deal
- **Clears Inventory** - Move slow-selling items as free gifts
- **Customer Perception** - "Free" is psychologically powerful
- **Flexible Promotions** - Many ways to structure deals

### Statistics

- BOGO offers can increase conversions by 25-50%
- "Free" items perform 20% better than equivalent discounts
- Multi-buy offers increase basket size by an average of 35%

## BOGO Types Explained

### Buy X Get Y Free

The classic BOGO - buy a certain quantity, get additional items free.

**Examples:**
```
Buy 2 Get 1 Free → Buy 2 shirts, get 3rd free
Buy 3 Get 1 Free → Buy 3 coffees, get 4th free
Buy 1 Get 1 Free → Classic BOGO
```

**Best for:**
- Consumables (food, beauty products)
- Items customers need multiples of
- Clearing inventory

### Buy X Get Y % Off

Buy items to unlock a percentage discount on additional items.

**Examples:**
```
Buy 1 Get 2nd 50% Off → Half price second item
Buy 2 Get 3rd 25% Off → Modest discount for third
Buy 1 Get 1 75% Off → Almost free second item
```

**Best for:**
- Higher-margin products
- Fashion items
- Trial encouragement

### Buy X Get Y $ Off

Fixed discount on the free item.

**Examples:**
```
Buy 2 Get 3rd $10 Off
Buy 1 Get 2nd $25 Off
Buy 3 Get 4th $50 Off
```

**Best for:**
- Expensive items
- Clear savings communication
- Consistent margins

### Spend X Get Y

Based on cart total, not item quantity.

**Examples:**
```
Spend $50 Get Free Shipping
Spend $100 Get Free Tote Bag
Spend $200 Get $20 Gift Card
```

**Best for:**
- Increasing cart value
- Gift with purchase promotions
- Holiday campaigns

## Creating a BOGO Offer

### Step 1: Basic Setup

1. Navigate to **BOGO** in sidebar
2. Click **Create BOGO Offer**
3. Enter offer details:
   - **Name**: Internal reference
   - **BOGO Type**: Select from dropdown
   - **Status**: Start as Draft

### Step 2: Configure the Deal

**For Buy X Get Y:**
- **Buy Quantity**: How many to buy (X)
- **Get Quantity**: How many free/discounted (Y)
- **Discount Type**: Free (100%), Percentage, or Fixed
- **Discount Value**: The discount amount

### Step 3: Product Selection

Choose which products participate:

- **All Products**: Store-wide BOGO
- **Specific Products**: Select individual items
- **Collection**: All items in a collection
- **Same Product**: Buy X of same product

### Step 4: Limitations

Set boundaries:

- **Max Uses Total**: Limit total redemptions
- **Max Uses Per Customer**: Per-customer limit
- **Start Date**: When offer begins
- **End Date**: When offer expires

### Step 5: Stacking Options

- **Stackable**: Can combine with other discounts
- **Exclusive**: Only this offer applies

## Display Settings

### How BOGO Shows to Customers

On product pages:
```
🎁 Buy 2, Get 1 FREE!
Add 3 items to cart to apply this offer
```

In cart:
```
Subtotal: $90.00
BOGO Discount (Buy 2 Get 1 Free): -$30.00
Total: $60.00
```

### Customization

- Badge text and color
- Cart message
- Success notification

## Advanced Features

### Auto-Add to Cart

When enabled, the free/discounted item automatically appears in cart when conditions are met.

### Mix and Match

Allow different products to count toward the buy quantity:

```
Buy ANY 2 shirts, get 3rd free
(Can be different styles/colors)
```

### Tiered BOGO

Increasing rewards:
```
Buy 2 Get 1 Free
Buy 4 Get 2 Free
Buy 6 Get 3 Free
```

## Best Practices

### 1. Clear Communication

Make the deal obvious:
```
✅ "Buy 2, Get 1 FREE!"
❌ "B2G1" (confusing abbreviation)
```

### 2. Protect Margins

Calculate break-even before launching:
```
Buy 2 Get 1 Free = 33% effective discount
Make sure margins support this
```

### 3. Set Limits

Prevent abuse:
- Max uses per customer
- Limited time window
- Specific products only

### 4. Test Performance

A/B test different configurations:
- Buy 2 Get 1 Free vs Buy 1 Get 1 50% Off
- Which drives more revenue?

## Troubleshooting

### Offer Not Showing

1. Check offer status (must be Active)
2. Verify product selection
3. Check date range
4. Clear cache

### Discount Not Applying

1. Verify cart meets conditions
2. Check for conflicting offers
3. Ensure correct products in cart
4. Verify quantity requirements

### Wrong Discount Amount

1. Review offer configuration
2. Check discount type (% vs $)
3. Verify "Get" quantity setting
