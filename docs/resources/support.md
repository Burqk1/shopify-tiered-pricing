## Getting the Most from Support

### 1. Be Specific

Instead of:
> "My discounts aren't working"

Try:
> "The 'Buy 2 Get 10% Off' rule on my T-Shirts collection isn't applying when I add 3 items to cart. I've verified the rule is active and the products are in the collection."

### 2. Include Screenshots

A picture is worth a thousand words:
- Screenshot of the issue
- Screenshot of rule configuration
- Screenshot of any error messages

### 3. Share Access if Needed

For complex issues, we may need temporary access:
- Create a staff account with limited permissions
- Share via secure channel
- We'll notify you when done

### 4. Test in Incognito

Before reporting, test in incognito/private browsing:
- Eliminates cache issues
- Removes logged-in state
- Cleaner debugging

## Bug Reports

Found a bug? Help us fix it!

### Report a Bug

1. Check [Known Issues](#known-issues) first
2. Email contact@novamentstudios.com with:
   - Bug description
   - Steps to reproduce
   - Expected vs. actual behavior
   - Screenshots/video
   - Browser console errors

### Bug Severity Levels

| Level | Description | Response |
|-------|-------------|----------|
| Critical | App unusable, data loss | Immediate |
| High | Major feature broken | Same day |
| Medium | Feature partially broken | 1-2 days |
| Low | Minor issue, workaround exists | Next release |
