# Changelog

All notable changes to Tiered Pricing Pro.

## Version 1.0.0 (January 2026)

🎉 **Initial Release**

### Features

#### Core Pricing Features
- **Volume Discounts** - Create quantity-based pricing tiers
- **Percentage Discounts** - Apply % off at tier thresholds
- **Fixed Amount Discounts** - $X off per item or total
- **Fixed Price Tiers** - Set exact prices per quantity

#### BOGO Offers (Growth+)
- Buy X Get Y Free
- Buy X Get Y at % Off
- Spend $X Get Y Free
- Mix and match product selection

#### Product Bundles (Growth+)
- Create product bundles with discounts
- Fixed price bundles
- Percentage discount bundles
- Frequently bought together widget

#### Cart Progress Bar (Growth+)
- Free shipping progress indicator
- Multi-tier goal system
- Customizable messaging
- Real-time updates

#### Free Gifts (Growth+)
- Automatic gift with purchase
- Spend threshold triggers
- Product-specific triggers
- Multi-tier gift system

#### Countdown Timers (Growth+)
- Fixed end date timers
- Recurring daily timers
- Evergreen (per-visitor) timers
- Multiple display styles

#### B2B/Wholesale (Growth+)
- Customer tag-based pricing
- Multi-tier customer levels
- Minimum order requirements
- Wholesale-only products

#### A/B Testing (Growth+)
- Price point testing
- Discount depth testing
- Statistical significance tracking
- Revenue impact analysis

#### Upsells & Cross-sells (Growth+)
- Product page recommendations
- Cart page suggestions
- Post-purchase offers
- AI-powered recommendations

#### AI Pricing (Professional)
- Smart price recommendations
- Demand-based suggestions
- Inventory optimization
- Confidence scoring

#### Geo Targeting (Professional)
- Country-based pricing
- Regional promotions
- Multi-currency support
- Location override option

#### Advanced Analytics (Professional)
- Comprehensive dashboard
- Rule performance tracking
- Customer segmentation
- Custom reports

#### Import/Export (Professional)
- Full configuration backup
- CSV bulk editing
- Store migration support
- API access

### Integrations
- Shopify POS support
- Shopify Functions for checkout

- Multi-language support (10 languages)

## How to Update

The app updates automatically. To check your version:

1. Go to Settings
2. Scroll to "About"
3. View current version

## Release Notes Notifications

Get notified about updates:

1. Go to Settings > Notifications
2. Enable "Product Updates"
3. Receive email on new releases
